1. Analytical Problem and Target User

As  an  Economics  and  Finance  student  learning  corporate  finance,  security analysis  and  fundamental  valuation  in  ACC102,  I  noticed  a  clear  gap  for individual retail investors: the evidence-based investment analysis framework we   learn    in   c lass    is   the    core   of    rational   stock    picking,    but   most non-professional investors cannot apply it easily. Traditional trading platforms only display scattered raw market and financial data, without integrating the full  fundamental  analysis  workflow  we  are  taught  into  a  simple,  intuitive interface. Many new investors make irrational decisions because they cannot translate  raw  numbers  into  actionable  investment  judgments,  or  miss  key financial risk signals. This tool is built to solve this problem. Its core purpose is to help non-professional individual investors quickly evaluate whether a listed company’s stock is worth buying, using the standard fundamental investment analysis framework from ACC102. The target  users  are  retail  investors  who want to conduct reliable, streamlined fundamental analysis without access to professional financial terminals.

2.  Dataset and Methodology

I used Yahoo  Finance as  my data source, which is a widely recognized free financial data platform approved in the assignment brief. I accessed the data via the yfinance Python library on April 22, 2026. The dataset includes real-time market valuation metrics, and the latest financial statement data from listed companies’ official SEC filings — the same data we use for investment case studies in our tutorials. I used Streamlit to build a no-code interactive interface, so users do not need any coding or professional finance background to run the analysis.  The  core  workflow  strictly  follows  the  fundamental  investment analysis framework from ACC102 lectures: data retrieval  → data cleaning  → core financial ratio calculation → peer benchmarking → quantitative downside risk assessment. I also added full exception handling to avoid invalid outputs from incorrect ticker inputs, which I refined after multiple rounds of testing and bug fixes.

3.  Key Outputs and Investment Insights 

The tool generates 4 core outputs to support data-driven investment decisions: core valuation indicators, historical price  trend,  de listing  and  financial  risk  warning,  and  peer-benchmarked financial ratio analysis. The key insight embedded in the tool, from my ACC102 lectures,  is  that  standalone  valuation  and  financial  ratios  are  meaningless without industry context. For example, a P/E ratio of 25 may be overvalued for the consumer staples sector, but fairly valued for the high-growth tech sector. This tool automatically completes this industry comparison, helping investors avoid the common mistake of judging ratios in isolation.

4.  Limitations and Learning
 
The tool has several key limitations. First, the risk assessment rules are simplified for basic investor screening, and it relies on Yahoo Finance’s data update frequency, which may have a short delay. Second, this tool is built exclusively for fundamental-focused value investors, and is not suitable for short-term traders who rely on technical analysis, intraday price movements,  or  market  sentiment  trading.  Third,  the  tool  can  only  provide standardized fundamental analysis reference, and cannot fully guarantee that investors   will   select   high-quality   stocks,    nor   can   it    replace   investors’ independent  judgment  and  comprehensive  due  diligence  before  making investment decisions. Future improvements could add a discounted cash flow (DCF)  valuation  module,  a  core  topic  in  our  corporate  finance  course, multi-year  financial   health  trend  analysis,   and  a   basic  technical   indicator module for users with different investment styles. Through this project, I did not just strengthen my Python and data analysis skills. I turned the textbook investment  analysis  frameworks  from  class   into  a   practical  tool  that   real investors can actually use. I spent hours fixing bugs, like handling empty data from invalid tickers and optimizing the peer comparison logic, and this process made me understand the core of fundamental investment analysis, as well as the boundaries of quantitative analysis tools, far better than just memorizing formulas.
